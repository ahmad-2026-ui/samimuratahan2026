# 🛍️ راهنمای نصب سیستم دوکان رخت‌فروشی (آفلاین)

## ✅ روش ۱: استفاده مستقیم از فایل HTML (ساده‌ترین)

### قدم ۱: دانلود فایل
فایل `dist/index.html` را دانلود کن و در کامپیوترت ذخیره کن.

### قدم ۲: اجرا
روی فایل `index.html` دوبار کلیک کن - در مرورگر باز می‌شود!

**⚠️ توجه:** اطلاعات در localStorage مرورگر ذخیره می‌شود. اگر کش مرورگر پاک شود، اطلاعات از بین می‌رود.

---

## ✅ روش ۲: نصب با Node.js (حرفه‌ای‌تر)

### قدم ۱: نصب Node.js
از سایت زیر Node.js را دانلود و نصب کن:
🔗 https://nodejs.org

### قدم ۲: دانلود پروژه
تمام فایل‌های پروژه را دانلود کن یا از GitHub بگیر.

### قدم ۳: نصب پکیج‌ها
در پوشه پروژه، ترمینال باز کن و بنویس:
```bash
npm install
```

### قدم ۴: اجرا
```bash
npm run dev
```

### قدم ۵: باز کردن
در مرورگر برو به: `http://localhost:5173`

---

## ✅ روش ۳: تبدیل به برنامه دسکتاپ با Electron

اگر می‌خواهی مثل یک برنامه واقعی Windows باشد:

### قدم ۱: نصب Electron
```bash
npm install electron --save-dev
```

### قدم ۲: ساخت فایل main.js
فایل `electron-main.js` که در پروژه ساختم را استفاده کن.

### قدم ۳: اجرا
```bash
npm run electron
```
